# 数据源为SogouC.reduced.rar.

`Sogou_webpage.mat` 存储有 `wordMat` 和 `doclabel` 两个变量。前者为特征矩阵，规模为 14400*1200，即包含了14400个文档关于1200个关键词的词频信息；后者为14400个文档的标签。关键词信息见keyword.xls，关键词按照词频从高到低降序排列，序号对应wordMat列序；标签1-9与实际标签对应如下：

1-	财经新闻  
2-	IT新闻  
3-	健康新闻  
4-	体育新闻  
5-	旅游新闻  
6-	教育新闻  
7-	招聘新闻  
8-	文化新闻  
9-	军事新闻
